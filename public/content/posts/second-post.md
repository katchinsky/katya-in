# Getting Started with Markdown

Markdown is a lightweight markup language that you can use to add formatting elements to plaintext text documents.

## Basic Syntax

### Headers

```
# H1
## H2
### H3
```

### Emphasis

*This text will be italic*
**This text will be bold**

### Lists

1. First item
2. Second item
3. Third item
